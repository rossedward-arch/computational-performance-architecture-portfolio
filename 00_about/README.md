# About This Portfolio

This portfolio showcases my work in **computational performance analysis, parametric and generative design, machine learning surrogates, and interpretable AI feedback for architecture**.  

It demonstrates my interdisciplinary skill set in Python, ML, building performance simulation, and architecture-led computational workflows.

## Contents Overview

- **Energy Simulation Projects:** Dynamic simulations of building energy performance.
- **Parametric & Generative Design:** Algorithmic form-finding and design space exploration.
- **Machine Learning Surrogates:** Surrogate models to accelerate performance predictions.
- **Interpretable AI Feedback:** Spatially-aware feedback and explainable AI for design reasoning.
- **Integrated Design Workflows:** Full end-to-end computational design pipelines.
- **Research Case Studies:** Applied mini-projects and experiments demonstrating methodologies.

## How to Use

Each folder contains **README.md files**, project scripts, notebooks, and diagrams. Follow the folder-specific instructions to explore projects, run scripts, or view results.
