# Energy Simulation Projects

This folder contains projects focused on **building performance simulation**, including EnergyPlus and OpenStudio workflows. Projects demonstrate:

- Dynamic thermal and energy simulations.
- Evaluation against comfort standards (CIBSE, Passivhaus).
- Early-stage design performance feedback.
